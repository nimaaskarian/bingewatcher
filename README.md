# Bingewatcher
A script, making the life of binge-watchers/multitaskers easier. Keeps track of series -- or anything else with a lot of episodes, like tutorials -- that you watching with a couple of files.

## Installation
1. Clone this project: `git clone https://github.com/nimaaskarian/bingewatcher`  
2. Put bw.py into your path (`cp bingewatcher/bw.py /bin/bw` or do a symlink: `ln -sr bingewatcher/bw.py /bin/bw`). or make an alias.

## Usage
- You may use this project in any UNIX-based operating system. Or even Windows! (not tested)
- use `-h` option to see help options.

## Uninstall
1. Remove from path
    - Remove alias definition from your rc file.
    - Or if you did the cp or symlink, use `rm /bin/bw`
2. Go where you did the `git clone https://github.com/nimaaskarian/bingewatcher` and `rm -rf bingewatcher`
3. yeah do it. Use [cpp-interpreter](https://github.com/nimaaskarian/cpp-interpreter) instead and start coding instead of bingewatching series and shows.
